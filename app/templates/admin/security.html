Как работает блокировка IP

Система определяет IP клиента по следующим источникам:

  • X-Forwarded-For - для прокси и балансировщиков
  • X-Real-IP - для nginx и других прокси
  • CF-Connecting-IP - для Cloudflare
  • Remote Address - прямое подключение
В локальной сети:
Каждое устройство имеет свой IP
(192.168.100.101, 192.168.100.102, и т.д.)
